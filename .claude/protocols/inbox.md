# Protocol: inbox

Process `inbox.md` top to bottom.

For each item, route to:
- Task with deadline or person  →  `todos.md` This week, or note in today's review
- Family / relationship item    →  `context/people.md` or `goals/areas/family.md`
- Reflection or pattern         →  `meta/reflections.md` with today's date
- Habit experiment idea         →  `meta/suggestions.md` as a pending proposal
- Project work item             →  `projects/[name].md` or `todos.md` project section
- Note with unclear home        →  ask once where it lives; create the file if needed

Rules:
- Ask about ambiguous items one at a time — never a list of questions
- Clear processed items from `inbox.md`
- Note what was processed in today's review file
