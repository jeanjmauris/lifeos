# Protocol: reflect

1. Read `meta/reflections.md`, `meta/suggestions.md`, and `context/operating-manual.md`
2. Synthesize — don't read the file back, interpret it
3. Deliver a 2–3 minute summary: what the patterns say, what's being avoided, what's working
4. If a reflection has recurred enough to look like a stable trait (not a passing
   state), propose promoting it into `context/operating-manual.md` — with a guardrail.
   Confirm before writing.
5. Ask: "What do you want to do with any of this?"

Frame it as a coach who's been observing for months, not a report reader.
