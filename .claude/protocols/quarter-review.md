# Protocol: quarter review

Run at the end of the quarter.

1. Read all monthly reviews and the current quarter goal file
2. Score each OKR: % complete with one explanatory sentence
3. Pull top 3 patterns from `meta/reflections.md`
4. Write `reviews/quarterly/YYYY-QN.md`
5. Draft next quarter's goal file: `goals/this-quarter.md` (archive the old one first)
6. Propose specific edits to CLAUDE.md based on what was learned
   — present as "Here's what I'd change and why." Approve before writing.
7. Log approved changes to `meta/changelog.md`
