# Protocol: evening

1. Read today's habit log from `habits/log/[today].md`
2. Ask one question: "How did today go?" — let response complete before doing anything
3. Update today's habit log with completions
4. Write `reviews/daily/[today].md` with:
   - Wins (even small ones)
   - What didn't happen and one honest sentence why
   - Energy level: low / medium / high
   - One carry-forward item for tomorrow
5. If `inbox.md` has more than 3 unprocessed items, process them now
