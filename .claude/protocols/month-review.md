# Protocol: month review

Run on the last day of the month.

1. Read all weekly reviews from this month
2. Score each life area 1–5 with one honest sentence each
3. Check progress against monthly milestones in `goals/areas/`
4. Write `reviews/monthly/YYYY-MM.md`
5. Ask: "Is there a habit that should be retired, added, or scaled up?"
6. Propose one edit to the Current Season block in CLAUDE.md if priorities have shifted
